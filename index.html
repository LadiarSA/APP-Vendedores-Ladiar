📥 Importar datos

Subí el Maestro de clientes (nombre, tier, canal, zona, coordenadas) y la Hoja de reparto (código de cliente + vendedor + frecuencia + secuencia por día). El sistema cruza ambos archivos por código de cliente.

⚠️ Al confirmar, esto reemplaza todos los datos de ruteo actuales por lo que resulte de este cruce. Para clientes que ya existen, se preserva el patrón de semanas (quincenal S1S3/S2S4, mensual) que ya tenían cargado.

📋 Exportar formato ERP

Genera el archivo Excel con la estructura requerida por el ERP: código de cliente, código de vendedor, patrón de días LMMJVSD, frecuencia y orden de visita.

Esta fecha define a qué Semana/Día del esquema de ruteo corresponde el día 1. A partir de ahí se calculan automáticamente las fechas de las visitas siguientes (cada Semana = +7 días).

⚠️ Si hay cambios manuales pendientes (pestaña Cambios), se exportan con su posición final ya aplicada.

También podés exportar una planilla simple de cobertura (Zona, Vendedor, Cliente, Tier, Frecuencia, Semana, Día), sin el formato técnico del ERP — útil para revisión y análisis.
